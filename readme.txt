== Description ==

Chehrak.com is free service for providing avatars.

On Chehrak, users can simply register with their email address and upload an avatar which is associated with their accounts. Also users can link multiple email addresses with the main account..

Chehrak.com will try to show avatars with some fixed square sizes starting from 32 pixels up to 256 pixels. so if the uploaded avatar is larger or smaller, the avatar is scaled appropriately. 

Avatars are loaded from the Chehrak CDN servers around the world using a URL containing an MD5 hash of the associated email address.

When you activate this plugin all avatar request just send to chehrak.com

== Installation ==

1. Upload `chehrak` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
